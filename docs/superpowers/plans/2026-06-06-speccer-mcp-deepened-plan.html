Speccer MCP: Implementation Plan & Deepened Tech Spec

For agentic workers: REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (- [ ]) syntax for tracking.

Goal: Build a high-performance Go-based Model Context Protocol (MCP) server that validates AI-generated engineer specs against actual Python workspaces using a hybrid Python AST compiler pipeline, checking for API and database drift, and scaffolding boilerplate models and routes securely.

Architecture: Go-based JSON-RPC over Stdio MCP server embedding a native Python AST parser. The Go server orchestrates workspace scans, parses Markdown AST layout associations, executes the Python parser in the workspace virtual environment, compares unified schemas, and outputs scaffold previews before safe file-writes.

Tech Stack: Go (1.20+), Python (3.8+ with native ast), yuin/goldmark for markdown AST parsing, github.com/google/uuid, and standard JSON-RPC over stdio.

System Architecture & Deepened Design Decisions

1. Go-Python Packaging & Distribution (Resolving Issue 2)

To eliminate deployment fragility, the Go server uses the standard library embed package to package the Python helper. On server startup, Speccer checks if the embedded Python helper is written to disk. If not, it safely unpacks speccer_parser/parser.py to a user global cache directory (e.g., ~/.cache/speccer/parser.py). All executions then target this cache directory script, ensuring zero external python dependency installations.

2. Static Import Resolution Boundaries (Resolving Issue 3)

The Python AST helper builds a static workspace symbol table. When parsing SQLAlchemy mixins or nested Pydantic models imported from other modules, Speccer matches import statements against mapped workspace files. To ensure absolute robustness, we enforce these boundaries:

  • Wildcard imports (e.g., from .models import *) are ignored for resolution; symbols defined in the same file are matched, and unresolvable external types gracefully fall back to "unresolved_type".
  • Relative imports (e.g., from ..schemas import UserBase) are resolved by computing the module path relative to the configuration's source_files globs.
  • Import aliases (e.g., import sqlalchemy as sa) are resolved dynamically in the symbol mapper.

3. Markdown Heading AST Association Algorithm (Resolving Issue 4)

Instead of rigid section-matching, Speccer traverses the yuin/goldmark AST. To associate tables and code blocks with specific contexts:

  1. Initialize a traversal context tracking the current nested heading state (H1, H2, H3).
  2. For each block node, check if it is a heading. If yes, extract its heading text and run the primary regex intent classifier to assign the active Intent Domain (e.g., data_schema, api_contract).
  3. Subsequent code blocks, markdown tables, or lists inherit the active intent domain of their closest parent/ancestor heading block.
  4. If an ancestor's intent domain is ambiguous (unclassified), the classifier aggregates the heading text and the first paragraph under it to determine the domain.

4. Safe-Write & Dry-Run Preview Protocol (Resolving Issue 5 & 6)

The scaffold_code tool is built around a dry-run preview protocol to protect non-preview-capable MCP clients. By default, scaffold_code does not write files to disk. Instead, it returns a preview payload of generated file stubs. To write files, clients must pass write_to_disk: true, and if files already exist, Speccer creates backups (e.g., .speccer_bak/) before doing safe, atomic writes.

All LLM caches and logs are directed to ~/.cache/speccer/ rather than the workspace root to prevent leaking credentials or cluttering git repositories.

Step 3: Implement Markdown AST parser with yuin/goldmark + GFM extension 
package ast

import (
    "bytes"
    "regexp"
    "strings"
    "github.com/yuin/goldmark"
    "github.com/yuin/goldmark/ast"
    "github.com/yuin/goldmark/extension"
    "github.com/yuin/goldmark/text"
)

type SpecChunk struct {
    Heading string
    Domain  string // data_schema, api_contract, system_prose
    Content string
}

func ClassifyHeading(text string) string {
    text = strings.ToLower(text)
    dbRegex := regexp.MustCompile(`(database|models|schema|entities|table)`)
    apiRegex := regexp.MustCompile(`(api|routes|endpoints|controllers|views)`)
    if dbRegex.MatchString(text) {
        return "data_schema"
    }
    if apiRegex.MatchString(text) {
        return "api_contract"
    }
    return "system_prose"
}

func ParseSpec(content []byte) ([]SpecChunk, error) {
    md := goldmark.New(
        goldmark.WithExtensions(extension.Table),
    )
    reader := text.NewReader(content)
    doc := md.Parser().Parse(reader)

    var chunks []SpecChunk
    var currentChunk *SpecChunk

    err := ast.Walk(doc, func(n ast.Node, entering bool) (ast.WalkStatus, error) {
        if !entering {
            return ast.WalkContinue, nil
        }
        if n.Kind() == ast.KindHeading {
            headingNode := n.(*ast.Heading)
            var headingText bytes.Buffer
            for child := headingNode.FirstChild(); child != nil; child = child.NextSibling() {
                if child.Kind() == ast.KindText {
                    headingText.Write(child.(*ast.Text).Segment.Value(content))
                }
            }
            hStr := strings.TrimSpace(headingText.String())
            domain := ClassifyHeading(hStr)
            if currentChunk != nil {
                chunks = append(chunks, *currentChunk)
            }
            currentChunk = &SpecChunk{
                Heading: hStr,
                Domain:  domain,
                Content: "",
            }
        } else if currentChunk != nil && n.Type() == ast.TypeBlock {
            lines := n.Lines()
            for i := 0; i < lines.Len(); i++ {
                line := lines.At(i)
                currentChunk.Content += string(line.Value(content)) + "\n"
            }
        }
        return ast.WalkContinue, nil
    })

    if err != nil {
        return nil, err
    }
    if currentChunk != nil {
        chunks = append(chunks, *currentChunk)
    }
    return chunks, nil
}

Step 3: Implement Python AST parser with static imports & symbol table 
import ast
import json
import sys
import os

TYPE_NORMALIZE_MAP = {
    "Integer": "int",
    "String": "string",
    "Text": "text",
    "Boolean": "bool",
    "Float": "float",
    "DateTime": "datetime",
    "UUID": "uuid",
    "int": "int",
    "str": "string",
    "bool": "bool",
    "float": "float",
}

class SymbolTable:
    """Cross-file symbol registry for static import resolution."""
    def __init__(self):
        self.symbols = {}
        self.aliases = {}   # e.g. "sa" -> "sqlalchemy"
        self.files = {}     # file_path -> {class_name: fields}

    def add_file(self, file_path, models):
        self.files[file_path] = models
        for name, model in models.items():
            self.symbols[name] = {"source_file": file_path, "fields": model["fields"]}

    def resolve_import(self, node, current_file):
        """Resolve an import node and return dict of {name: source_file} mappings.
        Returns a mapping of imported names to their source files, or None
        if the import cannot be resolved."""
        resolved = {}
        if isinstance(node, ast.Import):
            for alias in node.names:
                name = alias.asname or alias.name
                self.aliases[name] = alias.name
        elif isinstance(node, ast.ImportFrom):
            # Handle wildcard imports — skip gracefully
            if node.names and node.names[0].name == '*':
                return None
            module = node.module or ""
            # Resolve relative imports (e.g. ., ..)
            base_dir = os.path.dirname(current_file)
            level = node.level or 0
            if level > 0:
                parts = module.split(".")
                for _ in range(level - 1):
                    base_dir = os.path.dirname(base_dir)
                # Search source files for matching module
                for fpath in self.files:
                    rel = os.path.relpath(fpath, base_dir)
                    rel_no_ext = rel.replace(".py", "").replace(os.sep, ".")
                    if rel_no_ext == module or rel_no_ext.endswith("." + module):
                        for alias in node.names:
                            name = alias.asname or alias.name
                            resolved[name] = fpath
            else:
                # Absolute import — search configured source files
                parts = module.split(".")
                for fpath in self.files:
                    rel = fpath.replace(".py", "").replace(os.sep, ".")
                    if rel == module or rel.endswith("." + module):
                        for alias in node.names:
                            name = alias.asname or alias.name
                            resolved[name] = fpath
        return resolved or None

def normalize_type(type_node):
    if isinstance(type_node, ast.Name):
        raw = type_node.id
        return TYPE_NORMALIZE_MAP.get(raw, raw)
    elif isinstance(type_node, ast.Attribute):
        raw = type_node.attr
        return TYPE_NORMALIZE_MAP.get(raw, raw)
    elif isinstance(type_node, ast.Subscript):
        # Handles Optional[type] or Union[type, None]
        value = normalize_type(type_node.value)
        slice_val = normalize_type(type_node.slice)
        if value in ("Optional", "Union"):
            return f"optional({slice_val})"
        return f"{value}[{slice_val}]"
    return "unresolved_type"

def extract_models(tree, file_path=""):
    models = {}
    for node in ast.walk(tree):
        if isinstance(node, ast.ClassDef):
            is_model = False
            for base in node.bases:
                base_name = normalize_type(base)
                if base_name in ("Base", "SQLModel", "Model", "BaseModel"):
                    is_model = True
            
            if is_model:
                fields = {}
                for item in node.body:
                    if isinstance(item, ast.Assign):
                        for target in item.targets:
                            if isinstance(target, ast.Name):
                                f_name = target.id
                                f_type = "unresolved_type"
                                nullable = True
                                if isinstance(item.value, ast.Call):
                                    func_name = normalize_type(item.value.func)
                                    if func_name == "Column":
                                        for arg in item.value.args:
                                            f_type = normalize_type(arg)
                                        has_primary_key = False
                                        for kw in item.value.keywords:
                                            if kw.arg == "nullable" and isinstance(kw.value, ast.Constant):
                                                nullable = kw.value.value
                                            if kw.arg == "primary_key" and isinstance(kw.value, ast.Constant):
                                                has_primary_key = kw.value.value
                                        # primary_key=True implies nullable=False
                                        if has_primary_key:
                                            nullable = False
                                fields[f_name] = {"type": f_type, "nullable": nullable}
                    elif isinstance(item, ast.AnnAssign):
                        if isinstance(item.target, ast.Name):
                            f_name = item.target.id
                            f_type = normalize_type(item.annotation)
                            nullable = "optional" in f_type
                            fields[f_name] = {"type": f_type, "nullable": nullable}
                
                models[node.name] = {
                    "name": node.name,
                    "fields": fields
                }
    return models

def main():
    if len(sys.argv) < 2:
        print("Usage: parser.py  ...", file=sys.stderr)
        sys.exit(1)
    
    symtab = SymbolTable()
    results = {}
    
    for f_path in sys.argv[1:]:
        if os.path.exists(f_path):
            with open(f_path, "r", encoding="utf-8") as f:
                source = f.read()
            tree = ast.parse(source, filename=f_path)
            models = extract_models(tree, f_path)
            symtab.add_file(f_path, models)
            results.update(models)
    
    # Resolve cross-file imports: for each model, resolve imported base types
    for f_path in sys.argv[1:]:
        if os.path.exists(f_path):
            with open(f_path, "r", encoding="utf-8") as f:
                source = f.read()
            tree = ast.parse(source, filename=f_path)
            for node in ast.iter_child_nodes(tree):
                if isinstance(node, (ast.Import, ast.ImportFrom)):
                    symtab.resolve_import(node, f_path)
    
    # Write clean JSON IR to stdout wrapped in stream markers
    print("===SPECCER-JSON-START===")
    print(json.dumps(results))

if __name__ == '__main__':
    main()

package parser

import (
    "bytes"
    "encoding/json"
    "fmt"
    "os"
    "os/exec"
    "path/filepath"
    "strings"
    "speccer/speccer_parser"
)

func ResolveInterpreter(workspaceRoot string) string {
    candidates := []string{
        filepath.Join(workspaceRoot, ".venv", "bin", "python3"),
        filepath.Join(workspaceRoot, "venv", "bin", "python3"),
        filepath.Join(workspaceRoot, "env", "bin", "python3"),
    }
    for _, path := range candidates {
        if _, err := os.Stat(path); err == nil {
            return path
        }
    }
    return "python3" // fallback to system level
}

func RunParser(workspaceRoot string, pyFiles []string) (map[string]interface{}, error) {
    interpreter := ResolveInterpreter(workspaceRoot)
    
    // Ensure parser.py is written to a standard secure user cache directory
    userCache, err := os.UserCacheDir()
    if err != nil {
        userCache = os.TempDir()
    }
    speccerCache := filepath.Join(userCache, "speccer")
    if err := os.MkdirAll(speccerCache, 0755); err != nil {
        return nil, fmt.Errorf("failed to make secure cache dir: %w", err)
    }
    
    scriptPath := filepath.Join(speccerCache, "parser.py")
    if err := os.WriteFile(scriptPath, speccer_parser.ParserScript, 0644); err != nil {
        return nil, fmt.Errorf("failed to extract embedded parser: %w", err)
    }

    args := append([]string{scriptPath}, pyFiles...)
    cmd := exec.Command(interpreter, args...)
    cmd.Dir = workspaceRoot
    
    var stdoutBuf, stderrBuf bytes.Buffer
    cmd.Stdout = &stdoutBuf
    cmd.Stderr = &stderrBuf

    if err := cmd.Run(); err != nil {
        return nil, fmt.Errorf("python execution error: %s, err: %w", stderrBuf.String(), err)
    }

    stdoutStr := stdoutBuf.String()
    startIdx := strings.Index(stdoutStr, "===SPECCER-JSON-START===")
    if startIdx == -1 {
        return nil, fmt.Errorf("unexpected python output. logs: %s", stderrBuf.String())
    }

    jsonPayload := strings.TrimSpace(stdoutStr[startIdx+len("===SPECCER-JSON-START==="):])
    var ir map[string]interface{}
    if err := json.Unmarshal([]byte(jsonPayload), &ir); err != nil {
        return nil, fmt.Errorf("failed to parse json IR: %w", err)
    }

    return ir, nil
}

package diff

import (
    "fmt"
    "strings"
    "speccer/ast"
)

type DriftFinding struct {
    Severity string `json:"severity"`
    Section  string `json:"section"`
    Message  string `json:"message"`
}

// parseTableFields extracts field definitions from a GFM-style markdown table
// with columns: Field, Type, Nullable (case-insensitive header matching).
// Supports variations like | Field | Type | Nullable |.
func parseTableFields(content string) []map[string]string {
    lines := strings.Split(content, "\n")
    var headers []string
    var rows []map[string]string
    for _, line := range lines {
        line = strings.TrimSpace(line)
        if line == "" {
            continue
        }
        // Skip separator rows (|---|---|---|)
        if strings.HasPrefix(line, "|---") {
            continue
        }
        if strings.HasPrefix(line, "|") && strings.HasSuffix(line, "|") {
            raw := line[1 : len(line)-1]
            cells := strings.Split(raw, "|")
            trimmed := make([]string, len(cells))
            for i, c := range cells {
                trimmed[i] = strings.TrimSpace(c)
            }
            if headers == nil {
                headers = trimmed
            } else {
                row := make(map[string]string)
                for i, h := range headers {
                    if i < len(trimmed) {
                        row[strings.ToLower(h)] = trimmed[i]
                    }
                }
                if len(row) > 0 {
                    rows = append(rows, row)
                }
            }
        }
    }
    return rows
}

// AnalyzeDrift compares a spec chunk against workspace IR models generically.
// It parses markdown tables from the spec content and diffs them against
// all workspace models, reporting type mismatches, missing fields,
// and nullability differences.
func AnalyzeDrift(spec ast.SpecChunk, workspace map[string]interface{}) []DriftFinding {
    var findings []DriftFinding

    // Parse spec tables into field definitions
    specFields := parseTableFields(spec.Content)
    if len(specFields) == 0 {
        // No table found — skip (not a data_schema chunk with table content)
        return findings
    }

    // Compare against every model in workspace IR
    for modelName, modelData := range workspace {
        modelMap, ok := modelData.(map[string]interface{})
        if !ok {
            continue
        }
        fieldsRaw, ok := modelMap["fields"].(map[string]interface{})
        if !ok {
            continue
        }

        for _, specField := range specFields {
            fName := specField["field"]
            specType := specField["type"]
            specNullable := strings.ToLower(specField["nullable"]) == "true"

            if fName == "" {
                continue
            }

            actual, exists := fieldsRaw[fName]
            if !exists {
                findings = append(findings, DriftFinding{
                    Severity: "P1",
                    Section:  spec.Heading + " / " + modelName,
                    Message:  fmt.Sprintf("missing field '%s' defined in spec but absent in workspace model '%s'", fName, modelName),
                })
                continue
            }

            actualMap, ok := actual.(map[string]interface{})
            if !ok {
                continue
            }

            actualType, _ := actualMap["type"].(string)
            actualNullable, _ := actualMap["nullable"].(bool)

            // Type mismatch check (normalized types)
            if strings.ToLower(actualType) != strings.ToLower(specType) {
                findings = append(findings, DriftFinding{
                    Severity: "P1",
                    Section:  spec.Heading + " / " + modelName,
                    Message:  fmt.Sprintf("field '%s' type mismatch. Spec expects '%s', code uses '%s'", fName, specType, actualType),
                })
            }

            // Nullability mismatch check
            if actualNullable != specNullable {
                findings = append(findings, DriftFinding{
                    Severity: "P2",
                    Section:  spec.Heading + " / " + modelName,
                    Message:  fmt.Sprintf("field '%s' nullable mismatch. Spec nullable=%v, code nullable=%v", fName, specNullable, actualNullable),
                })
            }
        }
    }

    return findings
}

package scaffold

import (
    "fmt"
    "io"
    "os"
    "path/filepath"
    "strings"
)

type ScaffoldPreview struct {
    DryRun  bool   `json:"dry_run"`
    Path    string `json:"path"`
    Content string `json:"content"`
}

func GenerateStubs(modelName, engine string, writeToDisk bool, targetPath string) (*ScaffoldPreview, error) {
    content := fmt.Sprintf(`class %s(Base):
    __tablename__ = '%ss'
    # TODO: Define fields
`, modelName, strings.ToLower(modelName))

    if !writeToDisk {
        return &ScaffoldPreview{
            DryRun:  true,
            Path:    targetPath,
            Content: content,
        }, nil
    }

    // Backup existing file to .speccer_bak/ directory (per Section 4 spec)
    if _, err := os.Stat(targetPath); err == nil {
        bakDir := filepath.Join(filepath.Dir(targetPath), ".speccer_bak")
        if err := os.MkdirAll(bakDir, 0755); err != nil {
            return nil, fmt.Errorf("failed to create backup dir: %w", err)
        }
        bakPath := filepath.Join(bakDir, filepath.Base(targetPath)+".bak")
        src, err := os.Open(targetPath)
        if err != nil {
            return nil, err
        }
        defer src.Close()
        dst, err := os.Create(bakPath)
        if err != nil {
            return nil, err
        }
        defer dst.Close()
        if _, err := io.Copy(dst, src); err != nil {
            return nil, err
        }
    }

    if err := os.WriteFile(targetPath, []byte(content), 0644); err != nil {
        return nil, err
    }

    return &ScaffoldPreview{
        DryRun:  false,
        Path:    targetPath,
        Content: content,
    }, nil
}
